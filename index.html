HAN SHOT FIRST

Instructions:

  1. The first character you select gets thrown into the attacker area.
  2. The second character you select gets thrown into the defender area.
  3. Battle begins, use the "shoot" button to inflict damage on your opponent.
  4. If you successfuly defeat the first opponent, choose another until you are out of defenders.
  5. If you eliminate all defenders before running out of health, you win!!
  6. You can choose to reset the game using the "reset" button at any time.
